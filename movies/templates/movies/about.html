About

This project showcases the Marvel Cinematic Universe (MCU) movies and was developed using the Django framework. The web application features engaging animations for user interaction, implemented through a combination of CSS and JavaScript. Django modules are employed to efficiently store and manage the movie data, providing a seamless and organized experience for users navigating through the MCU movie collection
